Acerca deste projeto

O objetivo deste projeto é o desenvolvimento de um conjunto de páginas Web utilizando HTML, CSS e Javascript para a disciplina de Laboratório de Informática do curso do Mestrado Integrado em Engenharia de Computadores e Telemática, da Universidade de Aveiro.

Foram utilizados os seguintes recursos externos:
Ícones;
Bootstrap;
Opencv;
Imagens;
E material de Design.
