Deep Learning e Visão Computacional, duas tecnologias que se constituem a partir do uso de redes neurais. Redes essas que formam a base para a Visão Computacional, permitindo assim o processamento de imagens de maneira inteligente e eficaz. Tornando um processos que antes eram restrito a seres vivos, possível à máquinas.

Leia mais sobre DL e VC

Para um humano, é simples olhar para uma imagem e identificá-la, contudo, para um computador é necessário converter a imagem para algo mais simples. Tendo em vista que uma imagem não passa de uma matriz tridimensional que contém valores de Red, Green, Blue (RGB) o computador simplifica a imagem e a transforma em uma matriz de valores e será possível aplicar diversos filtros que possibilitam a detecção e compreensão da imagem. Tudo isto é possível através do uso de Redes Neurais. Sejam Recorrente (RNN) ou Convolucionais (CNN).
